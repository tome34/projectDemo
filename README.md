# projectDemo
项目框架,可直接应用于项目当中.
1,该框架包含mvp +mvp 两种模式,可根据页面的复杂性自由选择.
2,抽取常用的base基层以及常用的工具类.
3,网络框架采用retrofit2 + rxjava2.
